Betterer Runner

BettererFilePaths#

A list of file paths.

type BettererFilePaths = ReadonlyArray<string>;

BettererRunner#

type BettererRunner = {
queue(filePaths?: BettererFilePaths, handler?: BettererRunHandler): Promise<void>;
stop(force: true): Promise<BettererSummary | null>;
stop(): Promise<BettererSummary>;
};

Methods#

queue()#

Queue a Betterer test run on a given set of files. Multiple calls will be debounced.

Args:

  • filePaths?: BettererFilePaths - List of files to test with Betterer. If filePaths is undefined then all files will be tested.
  • handler?: BettererRunHandler - Callback to run when the queued run is complete.

Returns: Promise<void>

stop()#

Stop the runner, and return the most recent BettererSummary (or null if a run hasn't finished yet`).

Args:

  • force?: true - If force is passed, the Betterer run will be stopped and any errors will be ignored.

Returns: Promise<BettererSummary>

BettererRunHandler#

type BettererRunHandler = (summary: BettererSummary) => void;

Args:

  • summary: BettererSummary - The summary of the completed run.

Returns: void
